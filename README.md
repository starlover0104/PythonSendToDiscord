# PythonSendToDiscord
Python tool to send regular messages constantly via python

# Prequisities 
- Requires Python 3.6.x+ 
- Requests module 

# How to use
1. Open CMD
2. Point to the dir of the script
3. Type `pip install requests`
4. Run the python file by typing `python main.py` or  `py main.py`
